# 🚀 Deploy Rápido no Vercel

## Opção 1: Via GitHub (2 minutos)

### 1. Criar repositório no GitHub:
1. Acesse https://github.com/new
2. Nome: `protocolos-medgm`
3. Clique "Create repository"

### 2. Push do código:
```bash
cd ~/protocolos-medgm
git remote add origin https://github.com/SEU-USUARIO/protocolos-medgm.git
git push -u origin main
```

### 3. Deploy no Vercel:
1. Acesse https://vercel.com
2. Clique "New Project"
3. Importar do GitHub → protocolos-medgm
4. Clique "Deploy"
5. Pronto! URL gerada em 2 minutos

## Opção 2: Vercel CLI (se preferir)

```bash
npx vercel
# Seguir instruções na tela
# Login → Deploy → Pronto!
```

## URL Final

Depois do deploy você terá:
- https://protocolos-medgm.vercel.app
- Ou domínio customizado: scripts.medgm.com.br
