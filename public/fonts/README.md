# Fontes Gilroy

Adicione os arquivos de fonte Gilroy nesta pasta:

- `Gilroy-Light.woff2`
- `Gilroy-Regular.woff2`
- `Gilroy-Medium.woff2`
- `Gilroy-SemiBold.woff2`
- `Gilroy-Bold.woff2`

## Como obter as fontes

1. Baixe a família de fontes Gilroy
2. Converta para .woff2 (use [transfonter.org](https://transfonter.org))
3. Coloque os arquivos nesta pasta

## Alternativa

Se não tiver acesso às fontes Gilroy, o sistema usará automaticamente fontes do sistema como fallback (Inter, -apple-system, etc).
